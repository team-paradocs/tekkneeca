# tekkneeca
Assistive Robot for Total Knee Arthroplasty

## Installation

- Clone the package

        cd ~/<ros2_ws>/src
        git clone

- Install submodules.

        cd ~/<ros2_ws>/src/tekkneeca
        git submodule init
        git submodule update

- Build the packages

        cd ~/<ros2_ws>
        colcon build

- source environment

        source ~/<ros2_ws>/src/tekkneeca/install/setup.bash
